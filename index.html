Hidebar is a visibility control center. It gives you fast access to the collections, objects, and Mask modifiers you use most, without digging through the Outliner. Build your own toggle Panels, then hide or reveal scene elements directly from the 3D View sidebar.

Fast Toggles and Solo

Use one Panel to switch visibility while you work. Toggle character parts, isolate elements with Solo, and return to the full setup without opening the Outliner.

Behind each Panel is a snapshot-based system that remembers previous states, respects nested Collections, restores mixed visibility flags, and safely handles missing data. This helps you work around Blender’s visibility limitations and keeps complex scenes predictable.

Build Your Own Panels

Build your own Panels for a character, environment, product, or any other part of your workflow. Add only the controls you actually need, give them clear names, and export a Panel as a JSON file when you want to reuse it in another .blend file.
